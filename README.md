### Upute za izradu projekta

Potrebno je razviti informacioni sistem striktno prema zadanim instrukcijama, programskim jezicima kao i razvojnim okruženjima.

### Zadaci i upute: 
- Striktno definisane tehnologije i razvojna okruženja u kojima se radi projekat
- Baza podataka pomoću koje se realizuje projekat mora biti relaciona baza podataka
- Informacioni sistem mora biti funkcionalan i lijepo dizajniran
- Izbornici, dugmići, upozorenja i cijeli frontend mora biti na nekom od službenih jezika u Bosni i Hercegovini
- Sve aktivnosti u sklopu izrade projektnog zadatka potrebno je dokumentovati kroz Azure DevOps platformu i to raditi kontinuirano iz sedmice u sedmicu.
- Kompletan projekat dokumentovati kroz projektnu dokumentaciju od teoretskog do praktičnog dijela informacionog sistema
- Sve aplikacije koje se razviju moraju se upakovati u Docker slike koje je na kraju izrade projekta potrebno dostaviti zajedno sa Azure DevOps projektom te dokumentacijom projekta.

### Tehnologije za izradu projekta:
- Frontend i backend: .Net C# (dozvoljeno korištenje sljedećih razvojnih okruženja: Bootstrap, Tailwind CSS i slično).
- Baza podataka: Microsoft SQL server.
- Mobilna aplikacija: .Net MAUI.

### Napomene:
- Sistemi koji se razvijaju obavezno moraju biti informacioni.
- Sistem mora imati relacionu bazu podataka.
- Korisnički interfejs mora biti na nekom od službenih jezika u Bosni i Hercegovini.
- Projekat se mora pratiti kroz Azure DevOps i projektnu dokumentaciju te ove 3 stvari moraju da odgovaraju jedna drugoj.
- Na kraju izrade projekta, dostaviti i Docker slike kako bi se projekat mogao pokrenuti kroz Docker.
